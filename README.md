# Central Pathology Website

Official website for **Central Pathology Laboratory, Satna** built using **Python (Flask) + HTML/CSS/JavaScript**.

This repository contains the full website including:

* Homepage
* Diagnostic Services Page
* Health Packages Page
* About Page
* Contact Page
* Rotating Health Tips Banner
* SEO optimization for Google
* Local business structured data
* WhatsApp booking integration

---

# Website Features

* Responsive design for mobile and desktop
* Rotating health tips banner using JavaScript

# Installation

Clone the repository

```
git clone https://github.com/YOUR_USERNAME/central-pathology-website.git
```

Enter project directory

```
cd central-pathology-website
```

Install dependencies

```
pip install -r requirements.txt
```

Run the server

```
python backend/app.py
```

---

# Access Website

Open browser and visit

```
http://127.0.0.1:5000
```

# Health Tips Banner

The homepage includes a **rotating health tips banner** implemented with JavaScript.

Location:

```

```

Tips rotate every **6 seconds** with fade animation.
